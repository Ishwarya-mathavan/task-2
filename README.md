# task-2

Titanic Dataset - Exploratory Data Analysis
Project Overview

This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to uncover patterns and insights related to passenger survival. Various statistical summaries and visualizations are used to understand the relationships between different features such as gender, age, and passenger class.

Objectives
Understand the structure of the Titanic dataset.
Analyze passenger demographics.
Explore factors affecting survival.
Visualize key trends using charts and graphs.
Technologies Used
Python
Pandas
Matplotlib
Seaborn
Jupyter Notebook
Dataset

The project uses the Titanic dataset (train.csv), which contains information about passengers such as:

Passenger ID
Survival Status
Passenger Class
Name
Gender
Age
Fare
Embarked Location
Analysis Performed
1. Data Exploration
Dataset preview using head()
Data type inspection using info()
Statistical summary using describe()
2. Survival Analysis
Analysis of survival rates based on gender.
Visualization of survival distribution.
3. Age Distribution
Histogram showing the distribution of passenger ages.
4. Passenger Class Analysis
Distribution of passengers across different ticket classes.
Visualizations

The notebook includes:

Survival by Gender Count Plot
Age Distribution Histogram
Passenger Class Distribution Plot
